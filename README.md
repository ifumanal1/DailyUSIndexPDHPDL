
# Daily PDH PDL USA Indexes

Automated daily report of Previous Day High (PDH), Previous Day Low (PDL) 
and daily range for the four major US stock indices:
- S&P 500 (^GSPC)
- NASDAQ 100 (^NDX)
- Dow Jones Industrial Average (^DJI)
- Russell 2000 (^RUT)

PDFs are generated automatically every morning via Claude Routines
and committed to the folder: Daily PDH PDL USA Indexes/
